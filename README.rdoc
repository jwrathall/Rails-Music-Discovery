== README
My music app. The initial concept was to create a service that reminded the user when their fav band had an upcoming album.
I always seem to miss album releases. The app has sort of morphed into a reminder service plus a music discovery service too. 
It now shows artist concerts and comparable artists
etc. 

It uses musicbrainz.org as the search api and lastfm
bootstrap 3,
rails 4,
ruby 2.1.0,
postgres 9.3
xpath,
angular.js,
topcoat.io

API's
Musicbrianz, lastfm, bandsintown
